Starting a New Game

When you first enter the dungeon you'll want to equip your player. (See the Quick Reference Guide for more information) The green and blue stairs icon is the symbol for the stairs up dungeon object.

To Complete the Scenario:
The object of the game is to complete each scenario objective and leave the dungeon by using the stairs up located on level 1.
To Attack an Enemy:
Use CTRL + [Direction Key] to attack an enemy. You can set defaults for the direction keys in the Command Preferences menu.

Moving Around

To move your player around the map use a [Direction Key]. These can be configured (also required if you're starting out using a non-US keyboard layout) using the Command Preferences menu. For a US keyboard, the direction keys are centered by default around the "L" key like the arrows of a compass with "O" being equivalent to "North".

Opening a Door

The pink sections of wall are doors. Once open, they cannot be closed again. Some doors cannot be seen by default because they are hidden.

To Open a Door:
Use SHIFT + [Direction Key] to open a door.
To Search for a Door:
Use the [Search Key] to look for hidden doors. This key is S by default. You can set the search key in the Command Preferences menu.

Using Scenario Objects

There are a great many objects in each scenario that your player can interact with. For a list of standard Scenario Objects see the Quick Reference section. There is a save point seen in the picture. You can use this to save your scenario progress.

To Use a Save Point:
Use the [Scenario Object] key while standing on top of the save point. You can set the scenario object key using the Command Preferences menu.
